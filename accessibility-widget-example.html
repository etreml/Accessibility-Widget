Accessibility Widget - Konfigurierbares Barrierefreiheits-Tool

Grundlegende Verwendung

Das Accessibility Widget kann einfach mit Standard-Einstellungen initialisiert werden:

<!-- HTML -->
<script src="accessibility-widget/accessibility-widget.js"></script>
<link rel="stylesheet" href="accessibility-widget/accessibility-widget.css">

<script>
// Standard-Konfiguration
const widget = new AccessibilityWidget();
</script>

Konfigurierbare Optionen

Das Widget kann vollständig konfiguriert werden:

const widget = new AccessibilityWidget({
    // Position des Buttons
    position: 'bottom-right', // 'bottom-right', 'bottom-left', 'top-right', 'top-left'
    
    // Farben
    buttonColor: '#007bff',
    buttonHoverColor: '#0056b3',
    
    // Größen
    panelWidth: 300,
    buttonSize: 50,
    
    // Z-Index
    zIndex: 9999,
    
    // Automatisch anzeigen beim Laden
    showOnLoad: true,
    
    // Sprache
    language: 'de'
});

Verfügbare Funktionen

  • Hoher Kontrast: Ändert die Farben für bessere Lesbarkeit
  • Farben umkehren: Invertiert alle Farben auf der Seite
  • Links hervorheben: Zeigt alle klickbaren Elemente deutlich an
  • Lesehilfe: Fügt horizontale Linien zur besseren Orientierung hinzu
  • Textgröße: Vergrößert oder verkleinert den Text
  • Bewegung reduzieren: Deaktiviert Animationen für empfindliche Nutzer
  • Seite vorlesen: Liest den gesamten Seiteninhalt vor
  • Element vorlesen: Liest einzelne Elemente beim Klicken vor

Beispiel-Konfigurationen

Minimalistische Konfiguration

const widget = new AccessibilityWidget({
    position: 'bottom-left',
    buttonColor: '#28a745',
    showOnLoad: false
});

Großes Widget für Touch-Geräte

const widget = new AccessibilityWidget({
    buttonSize: 60,
    panelWidth: 350,
    buttonColor: '#ff6b35',
    buttonHoverColor: '#e55a2b'
});

Alle verfügbaren Positionen

// Oben
const widget1 = new AccessibilityWidget({ position: 'top-left' });
const widget2 = new AccessibilityWidget({ position: 'top-center' });
const widget3 = new AccessibilityWidget({ position: 'top-right' });

// Mitte
const widget4 = new AccessibilityWidget({ position: 'middle-left' });
const widget5 = new AccessibilityWidget({ position: 'middle-right' });

// Unten
const widget6 = new AccessibilityWidget({ position: 'bottom-left' });
const widget7 = new AccessibilityWidget({ position: 'bottom-center' });
const widget8 = new AccessibilityWidget({ position: 'bottom-right' });

Konfigurierbare Abstände

// Verschiedene Abstände vom Rand
const widget1 = new AccessibilityWidget({ margin: 10 });   // 10px vom Rand
const widget2 = new AccessibilityWidget({ margin: 30 });   // 30px vom Rand
const widget3 = new AccessibilityWidget({ margin: 50 });   // 50px vom Rand

// Kombiniert mit verschiedenen Button-Größen
const widget4 = new AccessibilityWidget({ 
    margin: 25, 
    buttonSize: 60 
}); // Größerer Button, angepasste Abstände

// Für Touch-Geräte mit größeren Abständen
const widget5 = new AccessibilityWidget({ 
    margin: 40, 
    buttonSize: 70 
}); // Großer Button, große Abstände

Mehrsprachige Unterstützung

// Englisch (Standard)
const widget1 = new AccessibilityWidget({ language: 'en' });

// Deutsch
const widget2 = new AccessibilityWidget({ language: 'de' });

// Automatischer Fallback auf Englisch bei unbekannten Sprachen
const widget3 = new AccessibilityWidget({ language: 'fr' }); // Wird Englisch verwenden

API-Methoden

Das Widget bietet verschiedene Methoden zur Steuerung:

// Widget anzeigen/verstecken
widget.show();
widget.hide();

// Widget komplett entfernen
widget.destroy();

// Globale Funktion für einfache Initialisierung
window.initAccessibilityWidget({
    position: 'bottom-right',
    buttonColor: '#007bff'
});

Integration in bestehende Projekte

Um das Widget in ein bestehendes Projekt zu integrieren:

  1. Laden Sie die Dateien accessibility-widget.js und accessibility-widget.css herunter
  2. Fügen Sie die CSS-Datei in den <head> ein
  3. Fügen Sie die JS-Datei vor dem schließenden </body> Tag ein
  4. Initialisieren Sie das Widget mit Ihrer gewünschten Konfiguration
<!-- In Ihrem HTML -->
<head>
    <link rel="stylesheet" href="accessibility-widget.css">
</head>

<body>
    <!-- Ihr Seiteninhalt -->
    
    <script src="accessibility-widget.js"></script>
    <script>
        // Initialisierung mit Ihrer Konfiguration
        const accessibilityWidget = new AccessibilityWidget({
            position: 'bottom-right',
            buttonColor: '#007bff',
            showOnLoad: true,
            language: 'de'
        });
    </script>
</body>
